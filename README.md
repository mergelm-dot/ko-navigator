# KO Navigator

KO Navigator ist eine professionelle Planungs- und Analyse-App für Knock-out-Zertifikate.

## Ziel

Die App unterstützt Nutzer bei der Vorbereitung und Planung von Trades.

KO Navigator soll:

- passende Knock-out-Zertifikate für den gewählten Broker finden
- Broker und Emittenten berücksichtigen
- Basiswerte über Name, Ticker, WKN und ISIN finden
- aktuelle Kurse, Währung und Referenzbörse automatisch laden
- den erwarteten Zertifikatspreis für einen geplanten Einstieg berechnen
- Hebel, Knock-out-Abstand und mögliche Risiken transparent darstellen

## Grundsatz

KO Navigator ist ein Werkzeug zur Entscheidungsunterstützung.

Die App gibt keine Kauf- oder Verkaufsempfehlungen und stellt keine Anlageberatung dar.

## Technischer Aufbau

Die App wird zunächst für Android entwickelt.

Die Architektur wird so vorbereitet, dass später:

- eine iPhone-Version
- eine Serveranbindung
- gemeinsame Kurs- und Zertifikatsdaten
- eine geschützte Berechnungsengine

möglich sind.
